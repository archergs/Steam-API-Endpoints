Official API? Name Description Example URL (excluding API key and SteamID) Parameters
Yes GetSteamLevel Returns the level of the users account http://api.steampowered.com/IPlayerService/GetSteamLevel/v1/?key=STEAM_API_KEY&steamid=USER_STEAM_ID key: API Key from Steam
steamid: SteamID of the user
Yes GetPlayerSummaries Returns a summary of the users account http://api.steampowered.com/ISteamUser/GetPlayerSummaries/v0002/?key=STEAM_API_KEY&steamids=USER_STEAM_ID key: API Key from Steam
steamids: Comma-delimited list of 64 bit Steam IDs (can take up to 100)
Yes GetRecentlyPlayedGames Returns the users recently played games (last 2 weeks) http://api.steampowered.com/IPlayerService/GetRecentlyPlayedGames/v0001/?key=STEAM_API_KEY&steamid=USER_STEAM_ID&format=json key: API Key from Steam
steamid: SteamID of the user
format: Output format. json (default), xml or vdf
count: Optionally limit the number of games returned
Yes GetOwnedGames Returns a list of all the games a user owns http://api.steampowered.com/IPlayerService/GetOwnedGames/v0001/?key=STEAM_API_KEY&steamid=USER_STEAM_ID&include_appinfo=1&format=json key: API Key from Steam
steamid: SteamID of the user (can take multiple)
include_appinfo: Also include game name and logo info
include_played_free_games: Also include free games that may not have been played
appids_filter: Filter the returned list to a set of appids. See offical docs for more info.

USER_STEAM_ID: SteamID of the user (can take multiple)
